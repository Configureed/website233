# BKA License System

## Installation

1. Install Python 3.11+
2. Run: `pip install -r requirements.txt`
3. Edit bot.py and add your Discord bot token
4. Run: `python bot.py`

## Commands
- `/gen 30` - Generate 30 day key
- `/keys` - List all keys
- `/info KEY` - Key details
- `/revoke KEY` - Revoke key
- `/banip IP` - Ban IP
- `/dashboard` - Update dashboard
